Body

Media type: application/json

Type: object

Properties
  • client_id: required(integer)

    Client's unique identification

    Example:

    1
  • amount: required(number)

    Loan amount in dollars

    Example:

    1000
  • term: required(number)

    Number of months that will take until the loan gets paid-off

    Example:

    12
  • rate: required(number)

    Interest rate as decimal

    Example:

    0.05
  • date: required(datetime)

    When the loan was requested (origination date as an ISO 8601 string)

    Example:

    2019-05-09T03:18:00Z

  • id: required(string)

Type: object

    Type of payment ("made" or "missed")

  • date: required(datetime)
